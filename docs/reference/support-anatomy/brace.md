# Brace

A brace is a support-to-support stabilizer. It connects one support shaft to another without touching the model.

## What it is

- A shaft with a knot at both ends.
- Used to stiffen or stabilize a support network.
- Can connect to trunks, branches, sticks, twigs, or other braces.

## Geometry

- End A: knot seated on a host shaft.
- Body: continuous brace shaft.
- End B: knot seated on another host shaft.
- Optional internal joints may be enabled by settings.

## Behavior

- A brace can connect to two points on the same support if the spacing rules allow it.
- Knots can be shared in some merge flows, or offset when kept separate.
- Dragging a knot slides the brace along the host shaft.

## Constraints

- Brace ends must never touch the model.
- Brace knots must not overlap unless they are intentionally merged.
- The brace must remain serviceable for edits and removal.
