# Quick Start

This is the fastest path from opening DragonFruit to exporting a supported model.

## 1) Open a model

- Import an STL (or supported scene input).
- Confirm the model appears in the scene.

## 2) Prepare the model

- Rotate and position as needed.
- Ensure the model sits correctly relative to the build plate.
- Use transform controls for fine adjustments.

## 3) Place supports

- Enter Support mode.
- Place trunks on required overhang areas.
- Add branches/auxiliary supports as needed for stability.

## 4) Enable raft (optional, recommended for many cases)

- Turn raft on in Support settings.
- Tune thickness, chamfer, and wall parameters.

## 5) Analyze and verify

- Run island analysis to inspect unsupported areas.
- Review support paths and contact quality.

## 6) Export

- Export the prepared geometry (STL and/or scene formats as needed).
- Validate in your printer workflow before production use.
