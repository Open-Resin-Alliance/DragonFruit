# Printer image assets

Store default printer images used by the preset library here.

Recommended structure:

- `public/assets/printers/concepts3d/athena2-12k.png`
- `public/assets/printers/concepts3d/athena2-16k.png`

Notes:

- JSON presets reference these paths as `/assets/printers/...`.
- PNG with transparent background works best for the profile cards.
- `Generic Machine` intentionally uses an in-app symbol (no image file needed).
